# nativebase extension README

NativeBase v3 snippets are a group of short-hands that can be used to further speed up your development. All snippets start with the prefix nb- followed by the name of the component, so that itâ€™s easy to find and use them.

## Features

This extension allows you to speed up your development by providing working snippets and easy navigation between different breakpoints using tabs.
